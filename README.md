# 本地 TXT 阅读器 Demo

一个面向 Windows 桌面的纯本地 TXT 小说阅读器。它用 Electron 把“导入 TXT、阅读、记进度、调排版”这几件事放在同一个离线应用里完成，不依赖账号、云同步或后端服务。

## 用途

- 本地导入 TXT 小说并建立书架
- 支持拖拽导入和文件弹窗导入
- 阅读时自动记录进度和上次位置
- 支持章节目录跳转和沉浸式阅读
- 支持字号、行距、字距、宽度、主题和字体切换

## 技术栈

- Electron
- Node.js
- 原生 HTML / CSS / JavaScript
- `fs` / `fs/promises`
- `iconv-lite`

## 运行方式

先安装依赖：

```powershell
npm install
```

启动应用：

```powershell
npm start
```

校验脚本：

```powershell
npm run check
```

## 部署方法

当前仓库的“部署”方式就是把项目放到目标 Windows 机器上直接运行：

1. 安装 Node.js
2. 获取这个仓库代码
3. 在项目目录执行 `npm install`
4. 执行 `npm start`

如果你要把它做成可分发的独立安装包，现在这个仓库还没有配置打包脚本；后续可以再接 `electron-builder` 或同类工具来生成安装程序。

## 本地数据

- 书籍文件会复制到 Electron 的 `userData` 目录
- 阅读进度、排版设置和书架数据也保存在本机
- 不上传、不同步、不依赖网络账号

## 当前功能

- 首屏书架
- TXT 导入
- 拖拽导入
- 阅读位置恢复
- 章节目录
- 排版设置
- 本地字体选择
- 进度保存

## 说明

这个项目定位是本地阅读 Demo，重点在阅读体验和离线可用性，不做云书城、账号体系或多格式支持。
